# 📚 Algoritmos 


Contiene ejercicios prácticos, trabajos, implementaciones de estructuras de datos y resolución de problemas utilizando Python, con el objetivo de reforzar conceptos fundamentales de programación y pensamiento algorítmico.

---

## 🎯 Objetivos

* Desarrollar lógica de programación.
* Aplicar técnicas de resolución de problemas.
* Implementar estructuras de datos básicas.
* Comprender el uso de funciones y recursividad.
* Practicar algoritmos de búsqueda y ordenamiento.

---

## 🛠️ Tecnologías Utilizadas

* Python 3
* Visual Studio Code
* Git & GitHub
